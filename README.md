# Servexa 🚀

### Smart Service Booking App

![Flutter](https://img.shields.io/badge/Flutter-Framework-blue)
![Firebase](https://img.shields.io/badge/Firebase-Backend-orange)
![Status](https://img.shields.io/badge/Status-Under%20Development-yellow)

Servexa is a modern **service booking mobile application** built using **Flutter & Firebase**.
It connects users with service providers and enables seamless booking, tracking, and management of services in real time.

---

## ✨ Features

### 👤 User App

* 🔍 Browse available services
* 📅 Book services with date & time
* 📍 Select location using map
* 📖 View booking history

### 🧑‍🔧 Provider App

* ✅ Accept / reject bookings
* 👥 View customer details
* 📊 Manage service requests
* 💰 Track earnings *(optional)*

### 🛠️ Admin Panel 

* 👤 Manage users & providers
* 📦 Monitor bookings
* ⚙️ Control platform data

---

## 🧰 Tech Stack

* **Flutter** – UI Development
* **Firebase Authentication** – User login
* **Cloud Firestore** – Database
* **Firebase Storage** – File storage
* **Google Maps API** – Location services
